电子商务平台接口合集

整合了淘宝，拍拍，京东三个主流平台的第三方商品管理接口
taobao.php 是主入口文件
t.php是接口功能文件
config.php是配置文件，主要放key和secret这些

可以用这套代码对电商的商品管理进行接口形式的底层支持，像库存更新的代码还优于“百胜”，你也可以基于这个形式对这套工具进行扩充，更好的支持你们自己家的商品管理
